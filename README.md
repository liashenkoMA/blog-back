# Blog Backend

Backend API для персонального блога.

Проект создал для практики разработки серверной части приложения: проектирование API, работа с базой данных, авторизация пользователей и бизнес-логика.

## 🚧 Status

Проект находится в разработке.

На текущем этапе реализовал базовый фундамент:

- структура NestJS приложения;
- подключение базы данных;
- основные модули;
- подготовка API для frontend.

Разработка временно остановлена.

Сейчас занят в другом проекте — приложении для изучения японского языка, где дополнительно тестирую:

- Docker;
- тестирование;

После завершения текущего проекта планирую вернуться к блогу, обновить архитектуру и продолжить разработку.
